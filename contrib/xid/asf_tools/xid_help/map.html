5.3 Mapping Functions

This panel lists available mapping functions. After selecting the desired mapping function, another panel will pop up listing the options available for that particular function. The mapping functions available are:
  1. Adjust - The adjust mapping function allows the user to interactively adjust the brightness and contrast of the displayed image.
  2. Load Mapping - The load mapping function allows the user to retrieve a mapping previously saved to disk.
  3. Piecewise linear - The piecewise linear mapping function builds a mapping with a piecewise linear fit from a set of mapping pairs (i.e., break points).
  4. Pseudocolor - The pseudocolor mapping function allows the user to assign specific colors to ranges of gray level values in black and white images.
  5. Save Mapping - The save mapping function allows the user to save a mapping to disk.
  6. Interactive - The interacitve mapping function allows the user to generate a piecewise linear mapping interactively by manipulating the histogram or mapping graph.
  7. Logarithmic - The logarithmic mapping function builds a logarithmic mapping in the specified base.
  8. Exponential - The exponential mapping function builds an exponential mapping in the specified base.
