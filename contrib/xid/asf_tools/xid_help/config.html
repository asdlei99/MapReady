8.0 General Information

8.1 Display Configuration

The configuration function is used to obtain information about the current display and the software version. Specifically, the configuration panel presents the following information:

  1. The current version of XID.
  2. The depth of the display (8-bit or 24-bit).
  3. The X11 visual being used (Pseudocolor or Directcolor).
  4. The width and height of the display.

Dismiss:
Close the configuration panel.
